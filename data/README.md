# 📂 数据获取说明

## 数据集来源

本项目使用的是"遥感解译大赛"官方数据集，包含高分二号（GF-2）卫星影像的光伏设施标注。

## 数据结构

数据采用 VOC 格式组织：

```
data/VOCdevkit/VOC2007/
├── JPEGImages/          # 原始卫星影像（.jpg）
├── SegmentationClass/   # 语义分割标签（.png）
└── ImageSets/           # 数据集划分文件
    └── Segmentation/
        ├── train.txt    # 训练集文件名列表
        └── val.txt      # 验证集文件名列表
```

## 获取方式

### 方式1：比赛官方渠道

如果比赛仍在进行或有公开数据集，请从官方渠道获取。

### 方式2：联系作者

如需获取数据集用于学术研究，请联系项目作者：
- 邮箱：your.email@example.com

### 方式3：自定义数据集

如果你想使用自己的数据，请按照以下格式准备：

1. **原始影像**：放入 `JPEGImages/` 目录，格式为 `.jpg`
2. **分割标签**：放入 `SegmentationClass/` 目录，格式为 `.png`
   - 背景像素值：0
   - 光伏设施像素值：1
3. **数据划分**：在 `ImageSets/Segmentation/` 中创建 `train.txt` 和 `val.txt`

## 数据预处理

准备好数据后，运行以下命令生成训练/验证集划分：

```bash
python src/prepare_data.py
```
